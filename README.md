# Description
Journal App built using Ruby on Rails

# Features
-User sign up and login.<br/>
-Create a Category for Tasks.<br/>
-Edit Category.<br/>
-Delete Category.<br/>
-Show all Categories.<br/>
-Create a Task in a Category.<br/>
-Edit a Task.<br/>
-Delete a Task.<br/>
-Show Task.<br/>
-See all Tasks of the Day.<br/>
