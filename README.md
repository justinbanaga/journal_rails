# Description
Journal App built using Ruby on Rails

# Features
-Create a Category for Tasks
-Edit Category
-Delete Category
-Show all Categories
-Create a Task in a Category
-Edit a Task
-Delete a Task
-Show Task
-See all Tasks of the Day
